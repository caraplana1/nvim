# Keybord Maping

## Moving

* `j`: Down
* `k`: up
* `l`: rigth
* `h`: left
* `<C-w> w`: Move to up window
* `<C-w> j`: Move to down window
* `<C-w> l`: Move to rigth window
* `<C-w> h`: Move to left window
* `<num> gg`: Move to number line
* `G`: Move to end of file

## View

* `F1`: coc-terminal
* `,v`: split screen vertical
* `,h`: split screen horizontal
* `<C-b>b`: open NERDTree
